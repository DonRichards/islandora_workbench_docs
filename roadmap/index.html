  • Roadmap (Fall 2021)
  • Edit on GitHub

Islandora Workbench development priorities for Fall 2021 are:

  1. ability to create taxonomy terms with fields (issue) - Done.
  2. ability to create hierarchical taxonomy terms (issue) - Done.
  3. ability to add/update remote video and audio (issue)
  4. ability to add/update multilingual field data (issue)
  5. ability to add/update a non-base Drupal title field (issue) - Done.
  6. allow --check to report more than one error at a time (issue)
  7. a desktop spreadsheet editor (issue)
  8. ability to add/update Paragraph fields (issue)
  9. support the TUS resumable upload protocol (issue)
